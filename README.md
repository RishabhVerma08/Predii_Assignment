# Vehicle Spec RAG

RAG (Retrieval-Augmented Generation) system for querying vehicle service manuals.

## Structure

- `data/`: Contains raw PDF manuals
- `config/`: Configuration settings
- `pdf_processing/`: PDF text extraction and chunking logic
- `vectorstore/`: Embeddings and vector database (ChromaDB) interaction
- `llm/`: Interaction with Gemini LLM
